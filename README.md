## Cofre Digital

Jogo simples em que o computador sorteia uma senha de 4 dígitos e você precisa descobrir cada dígito um por um. A cada chute errado, o jogo avisa se o número certo é maior ou menor.

### Como executar

```
python cofre_digital.py
```

### O que usei

- `random` pra sortear a senha
- `time` pra cronometrar quanto tempo levou
- `os` pra limpar o terminal

Fiz esse projeto pra praticar funções e loop `while` com condições mais elaboradas.
